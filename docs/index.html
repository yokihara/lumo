What it does

Brightness

Control the hardware backlight of external monitors over DDC/CI — including USB-C and HDMI displays.

Input switching

Change a monitor's input source from the menu bar, like a software KVM for a shared display.

Darkroom

Dim and warm any screen — including the built-in display — for a per-display night mode. Works even without DDC.
